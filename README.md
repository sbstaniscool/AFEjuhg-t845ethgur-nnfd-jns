## 🌟 Why this exists

The original **Art Class** project officially shut down and stopped all development on **March 2, 2025**.  
They announced:  

> *"Art Class has officially shut down and stopped development as of March 2nd, 2025. Thank you all for being a part of this. Feel free to fork this and host it yourself, but this repo will no longer be maintained."*

This **LTS (Long Term Support)** version picks up where they left off. We are committed to:

- 🛠️ **Fixing broken games** and patching vulnerabilities
- 🚫 Staying completely **ad-free** forever
- 🤝 Providing **ongoing maintenance** and community-driven updates
- 🚀 Finding **new ways to host** reliably (Note: Vercel is currently unsupported due to takedowns)

_No plans to monetize or add ads—pure community revival._

## 🚀 How to Self-Host

It's super easy to get Art Class running locally on your own machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 20+ Recommended)
- [Git](https://git-scm.com/)

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/walktheearth/Art-Class-Proxy.git
   cd Art-Class-Proxy
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the server:**
   ```bash
   npm start
   ```

4. **Open in your browser:**
   Navigate to `http://localhost:8080` (or whichever port is shown in the terminal output).
